structure.g4 - для визуализации через TestRig (C.g4 без семантики)
C.g4 - основной файл для ANTLR 4


gen.bat		- сгенерировать .py-файлы парсера и лексера
make.bat 	- скомпилировать все .c файлы из test_dir/source в .asm файлы в test_dir/compiled
open_tests.bat	- открыть скомпилированные файлы в SASM (IDE). Требует наличия установленного SASM, при необходимости измените путь к sasm.exe
clean.bat 	- очистить целевые папки


В /src находятся необходимые для работы компилятора скрипты:

name_table.py       - реализация таблицы имён
compile.py          - файл для запуска
graph_colouring.py  - раскраска графа
operation_utils.py  - вспомогательные элементы
tokens.py           - классы операций с отложенной генерацией кода

Туда же сохраняются файлы парсера и лексера при вызове gen.bat
